@author Sorrentino @plexus.component

Mojo Attributes:

  • Requires a Maven 2.0 project to execute.
  • Automatically executes within the lifecycle phase: site

Name Type Description
spaceKey String Confluence target confluence's spaceKey
checkoutDirectoryName String The directory name to checkout right after the scm url

Name Type Description

password String Confluence password
project MavenProject No description.
artifactMetadataSource ArtifactMetadataSource No description.
collector ArtifactCollector No description.
factory ArtifactFactory No description.
mavenProjectBuilder MavenProjectBuilder Maven Project Builder.
i18n I18N No description.
outputDirectory File No description.
anonymousConnection String The scm anonymous connection url. Default value is ${project.scm.connection}.
developerConnection String The scm developer connection url. Default value is ${project.scm.developerConnection}.
webAccessUrl String The scm web access url. Default value is ${project.scm.url}.
